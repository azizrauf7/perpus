aplikasi Perpustakaan Sekolah
Digital berbasis web yang bertujuan untuk memudahkan siswa dan admin perpustakaan dalam
peminjaman dan pendataan buku. Aplikasi ini dapat dipergunakan oleh seluruh warga sekolah serta
seluruh siswa/i sekolah tersebut. Aplikasi ini dibuat untuk menjawab tantangan digitalisasi tentang
peminjaman buku secara digital. Aplikasi ini berjalan tidak menggunakan internet, hanya berjalan
dengan koneksi lokal menggunakan Localhost.
